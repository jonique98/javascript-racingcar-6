
## 구조
```
+ src
  + constants
    └ constants.js --- constants를 선언한 파일
	+ racing
		└ RacingGame.js	-- game의 진행을 맡는 class
		└ Car.js	-------- 자동차 객체를 정의한 class
	+ view
		└ View.js	-------- 입출력을 관리하는 class
		└ Validation.js	-- 입출력 유효성을 관리하는 class
  └ App.js  ---------  프로그램 실행 진입점
	└ index.js  -------- App.play() 
```
